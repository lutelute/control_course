# 第16回：まとめ

## 1. 制御系設計の全体像

### 1.1 状態空間法の体系

```
システムモデリング → 解析 → 設計 → 実装
     ↓              ↓      ↓       ↓
  状態方程式      安定性   極配置   離散化
  伝達関数        可制御性  LQR    デジタル制御
                 可観測性  オブザーバ
```

### 1.2 主要な概念の関係

- **状態空間表現** ⟷ **伝達関数**（等価な表現）
- **極** ⟷ **固有値**（安定性を決定）
- **可制御性** ⟷ **可観測性**（双対関係）
- **LQR** ⟷ **カルマンフィルタ**（双対関係）

## 2. 各回の要点

### 第1回：状態空間表現
- $\dot{\mathbf{x}} = A\mathbf{x} + B\mathbf{u}$, $\mathbf{y} = C\mathbf{x} + D\mathbf{u}$
- 多入力多出力、内部状態の記述

### 第2回：PID制御
- 古典制御の代表、産業界で広く使用
- $u = K_p e + K_i \int e + K_d \dot{e}$

### 第3回：状態方程式の解
- 遷移行列 $e^{At}$、畳み込み積分
- $\mathbf{x}(t) = e^{At}\mathbf{x}_0 + \int_0^t e^{A(t-\tau)}B\mathbf{u}(\tau)d\tau$

### 第4回：遷移行列の計算
- ラプラス変換、対角化、ケーリー・ハミルトン、級数展開

### 第5回：伝達関数と状態空間表現
- $G(s) = C(sI-A)^{-1}B + D$
- 正準形（可制御、可観測、対角）

### 第6回：安定性と極
- $\text{Re}(\lambda_i) < 0$ ⟺ 漸近安定
- ラウス・フルビッツ判別法

### 第7回：リアプノフ方程式
- $V(\mathbf{x}) = \mathbf{x}^T P \mathbf{x}$ が減少 → 安定
- $A^T P + PA = -Q$

### 第8回：可制御性
- 可制御性行列 $\mathcal{C} = [B \; AB \; \cdots]$
- $\text{rank}(\mathcal{C}) = n$ ⟺ 可制御

### 第9回：可観測性
- 可観測性行列 $\mathcal{O} = [C; CA; \cdots]^T$
- $\text{rank}(\mathcal{O}) = n$ ⟺ 可観測

### 第10回：状態フィードバックと極配置
- $\mathbf{u} = -K\mathbf{x}$ で極を任意に配置
- アッカーマンの公式

### 第11回：オブザーバ
- $\dot{\hat{\mathbf{x}}} = A\hat{\mathbf{x}} + B\mathbf{u} + L(\mathbf{y} - C\hat{\mathbf{x}})$
- 分離定理

### 第12回：最適レギュレータ
- $J = \int (\mathbf{x}^T Q \mathbf{x} + \mathbf{u}^T R \mathbf{u}) dt$ を最小化
- 最適ゲイン $K = R^{-1}B^T P$

### 第13回：リッカチ方程式
- $A^T P + PA - PBR^{-1}B^T P + Q = 0$
- LQR のロバスト性

### 第14回：カルマンフィルタ
- 最適状態推定
- $L = PC^T R_v^{-1}$

### 第15回：サーボシステム
- 目標値追従
- 内部モデル原理、積分動作

## 3. 設計フローチャート

```
1. システムモデリング
   └─ 物理法則から状態方程式を導出

2. システム解析
   ├─ 安定性（固有値）
   ├─ 可制御性（極配置可能か）
   └─ 可観測性（状態推定可能か）

3. 制御器設計
   ├─ 極配置法（応答速度を指定）
   └─ LQR（最適性を考慮）

4. オブザーバ設計（状態が測れない場合）
   ├─ ルーエンバーガーオブザーバ
   └─ カルマンフィルタ（ノイズがある場合）

5. サーボ系設計（目標値追従が必要な場合）
   └─ 積分器の追加

6. 実装
   ├─ 離散化
   └─ デジタル制御
```

## 4. 重要な定理・公式

| 項目 | 公式・条件 |
|-----|----------|
| 安定性 | $\text{Re}(\lambda_i) < 0$ |
| 可制御性 | $\text{rank}[B \; AB \; \cdots] = n$ |
| 可観測性 | $\text{rank}[C; CA; \cdots]^T = n$ |
| LQRゲイン | $K = R^{-1}B^T P$ |
| カルマンゲイン | $L = PC^T R_v^{-1}$ |
| 遷移行列 | $\Phi(t) = e^{At}$ |
